# **App Name**: SignAI Website

## Core Features:

- Informational Landing Page: A landing page providing comprehensive information about SignAI, including its functionality, benefits, applications across sectors, and compliance features, tailored for hospitals, universities, and local governments in Peru.
- Lead Capture Form: Integrated HubSpot form for capturing leads, pre-filled with UTM parameters from the URL, triggering a 'lead_submitted' conversion event upon submission, with Google Analytics 4 support and console logging fallback.
- Interactive Demo Showcase: Showcase a compelling 90-second demo highlighting the wearable's real-time sign language translation capabilities, to provide potential clients with an engaging overview.
- Client Testimonials Display: Dynamic display of client testimonials and institutional logos, building trust and credibility with potential customers in the target sectors.
- AI-Powered Language Translation Explanation: Explain AI translation by illustrating how the wearable device's IMU sensors and computer vision work together to capture hand and facial expressions, and convert them into real-time text and voice using machine learning models. Highlight on-device processing and anonymization techniques.

## Style Guidelines:

- Primary color: '#176BFF', a strong blue, should visually align with modern tech and trust. The desaturated background color is '#E5F0FF'.
- Accent color: '#17B2FF' - a lighter blue (analogous hue, similar saturation, lighter) should be used sparingly for highlights.
- Font choice: 'Inter', a sans-serif font, should be used for both headlines and body text. Note: currently only Google Fonts are supported.
- Ensure a responsive layout that adapts seamlessly to different screen sizes, providing an optimal viewing experience across devices.
- Utilize clean and modern icons to represent key features and benefits, enhancing visual appeal and comprehension.
